About

A heavily optimized server fork via experimental patches

Threaded Dimensions

Canvas introduces a new Threaded Dimensions patch, which isolates the tick loop of each world onto its own "main thread", and isolating them to reduce sharing data, and improve performance in terms of scalability across worlds.

Faster Chunk Generation

Canvas implements a ton of chunk generation optimizations, improving chunk generation and loading performance

More Optimizations

Canvas optimizes most common places of lag in the Minecraft server, from villagers(now more than 2x faster), to hoppers(also being more than 2x faster), and more!
The benchmarks shown are tested on:
32.0 GB - Intel(R) Core(TM) i9-14900HX   2.20 GHz - Windows 11, with the villagers test run on 16.0GB - Intel(R) Core(TM) i5-6200U - Arch linux

Absurdly Fast

Canvas introduces tons of optimizations to keep your server insanely fast and stable.

Why Not Use Folia Instead?

Canvas is made to be more of an in-between for Purpur and Folia, offering more scalability and huge performance increases, while also maintaining better plugin compatibility, unlike Folia

Active Development

Canvas is still in active development, getting faster and more stable each day.
